


版本说明：


1.  版本号:   Ver 1.0.0

2.  C 编译器：  AVR-GCC 4.3.2 (winavr 20081205)

3.  功能描述:
	
		主机扫描到按键按下后，发送当前键盘值给接收器，如果发送成功，主机LED均亮

	否则，LED均灭；接收器接收到数据后，根据收到的键盘按键值判断如果按键按下，

	则灯均亮，否则均灭

4.  RF配置：
	
	4.1  频道： 40(十进制)
	
	4.2  收发地址： 0xb2, 0xb3, 0xb4, 0xb5, 0x03
	
	4.3  传输速率：  250KBPS
	
	4.4  载波长度： 5
	
	4.4  载波格式：  0x27, 按键值(1byte),  保留（2字节），校验和(1byte).
	

4.  保留
